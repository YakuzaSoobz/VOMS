CUSTOMER QUOTE MANAGER


This is a view of the Customer Quote Manager Page.

The Customer Quote Manager allows users to view, edit, create and sort Customer Quotes and invoices. This subsystem will allow employees to create a final quotation for a customer.Users can enter product details, alter prices, add a discount as well as print quotations or invoices.

The VIEW QUOTE button shows the user the actual quotation that has been selected.

The user has the option of printing the quotation by clicking on the PRINT CUSTOMER QUOTE button.


Printing popup.

__________________________________________________________________________________________________

A new customer quotation can be created by clicking on the CREATE NEW CUSTOMER QUOTE button.


This page is shown when the user decides to create a new customer quote.

Before creating the actual quotation, the user is required to fill out the quotation details and the save the quotation.


Quote reference ID is filled in automatically.

Customers can be selected by choosing their Customer ID from a drop down list, or by clicking on the 'i' symbol and choosing the customer from a table, as shown below.


Selecting Customer ID.


Selecting customer from the customer table.

If the customer is not stored on the system, there is also an option to create a new customer by clicking on the CREATE CUSTOMER button.
When the CREATE CUSTOMER button is selected, the Customer Manager page will open. (Refer to Customer Manager page of help system.)


Customer Manager page.


After selecting a customer, the employee who is responsible for creating the quotation can be selected in the same manner.
i.e. With the use of Employee ID or selecting from a table.

The date of the creation of the quotation needs to be selected. By default, the current date is shown.

A selection needs to be made about whether the quotation has been accepted by the customer or not. (Whether the customer is going to accept the items in the quotation as is.)
If the customer is accepting the quotation, the flag needs to be set as Y and the date of acceptance needs to be given. By default the date is set to the current date.


Once all of the required details have been filled in, the user needs to click on the SAVE QUOTE button.
The option to add items to the quotation is now available by clicking on the ADD ITEMS button.


The Customer Quote Line Item Manager is now opened.

Users can select a Supplier Quote from which the desired products are part of, then by clicking on the ADD buttton within the Supplier Quote Line Items table.



The Line Item Details page is now shown.

Within the Line Item Details page, the user can change certain details, such as the quantity, selected markup, selected discount, as well as change the VAT percentage.
The VAT percentage is stored permanently until a user changes the percentage.

Once the user is satisfied with the details, the SAVE button should be clicked in order to save the item to the quotation.


The Customer Quotation is now saved onto the database and can be accessed at any time.
