VENDOR ORDER MANAGEMENT SYSTEM

The Vendor Order Management System(VOMS) is a bespoke Front End System created for Mhlathuze Industrial Suppliers and Services cc. by Normalisers42.
The VOMS serves as a system that can be used by employees and managers to carry out all of the required business tasks in order to complete the business cycle.

_____________________________________________________________________________

Users access the system by logging in, using their email address and password, and by selecting their access level.
    Level 1    -    Employee
    Level 2    -    Manager


Sign in page.
_____________________________________________________________________________

Logging in as an employee (Level 1 access):


Employee home menu.

When logged into the VOMS as an employee, users have access to multiple features. Access to these features are gained by clicking on the icons, as shown above.

Customer Quote Manager:
        The customer quote manager is used to view, edit, archive and create new customer quotes.
Supplier Quote Manager:
        The supplier quote manager is used to view, edit, archive and create new supplier quotes.
Product Manager:
        The product manager is used to view, edit, archive and create new products.
Supplier Manager:
        The supplier manager is used to view, edit, archive and create new suppliers.
Customer Manager:
        The customer manager is used to view, edit, archive and create new customers.
 
_____________________________________________________________________________

Logging in as a Manager (Level 2 access):


Manager home menu.

When logged into the VOMS as a manager, along with the options that an employee has, following options are added as well:

Employee Manager:
        The employee manager is used to view, edit, archive and create new employees.
Report Menu:
        Managers can create and view reports that will track trends and be beneficial to decision making.
